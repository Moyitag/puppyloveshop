# PuppyLoveShop

## Descripción

PuppyLoveShop es una aplicación web desarrollada para facilitar la administración de una tienda de mascotas. El sistema permite gestionar de manera eficiente la información relacionada con productos, clientes y empleados mediante una interfaz intuitiva y una arquitectura moderna basada en tecnologías web.

El objetivo principal del proyecto es optimizar los procesos administrativos del negocio, proporcionando herramientas que permitan realizar operaciones de registro, consulta, actualización y eliminación de información de forma segura y organizada.

---

## Características

- Gestión de productos.
- Administración de empleados.
- Gestión de clientes.
- Interfaz responsiva para diferentes dispositivos.
- Operaciones CRUD para cada módulo.
- Consumo de servicios mediante API REST.
- Validación de formularios.
- Navegación dinámica entre módulos.

## Instalación

Clonar el repositorio:

```bash
git clone https://github.com/Moyitag/puppyloveshop.git
```

Ingresar al directorio del proyecto:

```bash
cd puppyloveshop
```

### Frontend

Instalar dependencias:

```bash
npm install
```

Ejecutar el proyecto:

```bash
npm run dev
```

### Backend

Ingresar al directorio correspondiente:

```bash
cd backend
```

Instalar dependencias:

```bash
npm install
```

Iniciar el servidor:

```bash
npm start
```
